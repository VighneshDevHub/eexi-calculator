Methodology

How the linear interpolation is derived and applied.

The straight-line equation

A straight line through (x₁, y₁) and (x₂, y₂) satisfies:

y = [(x₂ − x) / (x₂ − x₁)] · y₁ + [(x₁ − x) / (x₁ − x₂)] · y₂

To verify: substitute x = x₁ to get y = y₁, and x = x₂ to get y = y₂. If a linear fit is correct at two different points, it is correct everywhere along the segment.

Solving for y₃

Setting x = x₃ in the straight-line equation gives:

y₃ = [(x₂ − x₃)·y₁ + (x₃ − x₁)·y₂] / (x₂ − x₁)

Solving for x₃

Rearranging the y₃ formula to isolate x₃:

x₃ = [(y₂ − y₃)·x₁ + (y₃ − y₁)·x₂] / (y₂ − y₁)

Solving for the other four variables

The same equation is rearranged algebraically for each unknown:

Solve forFormula
y₁y₁ = [y₃·(x₂−x₁) − (x₃−x₁)·y₂] / (x₂−x₃)
y₂y₂ = [y₃·(x₂−x₁) − (x₂−x₃)·y₁] / (x₃−x₁)
x₁x₁ = [x₂·(y₁−y₃) + x₃·(y₂−y₁)] / (y₂−y₃)
x₂x₂ = [x₁·(y₃−y₂) + x₃·(y₂−y₁)] / (y₃−y₁)

Constraints

  • x values (x₁, x₂, x₃) must all be unique — they represent distinct positions on the x-axis.
  • y values (y₁, y₂, y₃) must all be unique — required to avoid division by zero when solving for x unknowns.
  • Exactly one field must be left empty — the tool solves for that one unknown.
  • The formula is valid only within the linear (first-order) approximation regime. For non-linear data, use higher-order interpolation (cubic spline, polynomial).
