Help and details

A straight line through (x₁, y₁) and (x₂, y₂) satisfies:

y = [(x₂ − x) / (x₂ − x₁)] · y₁ + [(x₁ − x) / (x₁ − x₂)] · y₂

Setting x = x₃ gives the solution for y₃, and rearranging solves for any of the six variables:

Solve forFormula
y₃ y₃ = [(x₂−x₃)·y₁ + (x₃−x₁)·y₂] / (x₂−x₁)
x₃ x₃ = [(y₂−y₃)·x₁ + (y₃−y₁)·x₂] / (y₂−y₁)
y₁ y₁ = [y₃·(x₂−x₁) − (x₃−x₁)·y₂] / (x₂−x₃)
y₂ y₂ = [y₃·(x₂−x₁) − (x₂−x₃)·y₁] / (x₃−x₁)
x₁ x₁ = [x₂·(y₁−y₃) + x₃·(y₂−y₁)] / (y₂−y₃)
x₂ x₂ = [x₁·(y₃−y₂) + x₃·(y₂−y₁)] / (y₃−y₁)

The x values must be unique and the y values must be unique. Exactly one box must be empty before clicking Calculate.
